O que é Java:

Java é uma linguagem de programação que possibilita escrever o código apenas uma vez e rodá-lo em diferentes dispositivos, passando a ser implementada em praticamente qualquer lugar, desde sites e computadores até datacenters, celulares, calculadoras, videogames, etc. Algo engraçado do logo desse programa é clássica imagem de uma caneca de café quente, uma bebida bastante comum no nosso dia-a-dia.
