O que é o JavaScript:

JavaScript é uma linguagem de programação com o propósito de oferecer aos desenvolvedores formas de tornar determinados processos de páginas web mais dinâmicos, tornando seu uso mais agradável. MAS, não se confunda! O JavaScript não é a mesma coisa que o Java, linguagem de programação desenvolvida pela Sun Microsystems.
